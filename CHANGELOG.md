# Changelog

- `README.md`, `DISCLAIMER.md`, `LICENSE` (MIT), and `SKILL.md`.
- `requirements.txt` dependency manifest.
- Community and collaboration files:
  - `CONTRIBUTING.md`
  - `CODE_OF_CONDUCT.md`
  - GitHub issue templates and pull request template
- GitHub Actions CI workflow for compile and CLI smoke checks.
