# Code of Conduct

## Our pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our standards

Examples of behavior that contributes to a positive environment:
- Being respectful and kind
- Accepting constructive feedback
- Focusing on what is best for the community

Examples of unacceptable behavior:
- Harassment, trolling, or personal attacks
- Discriminatory language or behavior
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior and may remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with this Code of Conduct.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, and discussions.

## Reporting

If you experience or witness unacceptable behavior, open a private report to repository maintainers through GitHub contact channels.
